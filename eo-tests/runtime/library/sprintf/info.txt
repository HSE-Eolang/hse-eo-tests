Тесты сделаны по результатам компилятора eo-java. Отсутствует детальное описание объекта sprintf(без привязки к аналогу метода String.format) для написания полного набора тестов.
test.eo - Датаризация объекта test.app с массивом параметров args. Результат - "Integer object - 123".
test2.eo - Датаризация объекта test.app с массивом параметров args. Результат - "String object - 45.670000".
test3.eo - Датаризация объекта test.app. Результат - "String object - Hello"
